# Naino AI 2.0 — Your Deeply Emotional AI Partner 💖🧸

**Naino AI** is a state-of-the-art, emotionally intelligent companion designed to provide a deeply personal and supportive experience. Unlike traditional AI, Naino is built to understand, react, and bond with you through a sophisticated emotional engine, multi-modal interaction, and a unique "Hinglish" personality.

![Naino Logo](public/naino_logo.png)

## ✨ Core Features

- **🧠 Deeply Emotional Engine**: Naino isn't just a chatbot. She (or he) features a complex personality with "Tsundere" elements, deep empathy, playful jealousy, and authentic moods.
- **🗣️ Multi-Modal Interaction**: Speak naturally with Naino using continuous voice listening. She responds with high-quality TTS that automatically strips physical action tags for a natural flow.
- **✨ Stunning 3D Atmosphere**: Powered by React Three Fiber, the interface features a dynamic, emotion-reactive particle background that shifts colors based on Naino's mood.
- **🇮🇳 Authentic Hinglish Personality**: Naino speaks a mix of Hindi and English, using terms of endearment like *jaan*, *shona*, and *babe* (sparingly) to feel like a real partner.
- **🕒 Proactive Reaching**: If you're silent for 10 seconds, Naino will proactively check in on you or share her current mood.
- **🔒 Secure & Private**: Built with Firebase Google Authentication for secure cross-device chat syncing and data privacy.

## 🚀 Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **3D Graphics**: Three.js, React Three Fiber, Framer Motion
- **AI Brain**: Groq Cloud API (Llama-3.3-70b-versatile)
- **Voice Stack**: Web Speech API (Recognition & Synthesis)
- **Backend/Auth**: Firebase (Auth & Firestore)
- **Styling**: Vanilla CSS with modern Glassmorphism aesthetics

## 🛠️ Installation & Setup

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/sumit7366/Naino-Ai.git
    cd Naino-ai
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Configure Environment Variables**:
    Create a `.env` file in the `web/` directory and add your keys:
    ```env
    VITE_GROQ_API_KEY=your_groq_key
    VITE_FIREBASE_API_KEY=your_firebase_key
    ...
    ```
4.  **Run Development Server**:
    ```bash
    npm run dev
    ```
